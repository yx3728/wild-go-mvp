# Wild Go MVP

Mobile-first prototype for Wild Go, an urban nature collection app where real organism photos become rarity-based collectible cards.

## Run

```bash
npm install
npm run dev -- --port 5173
```

Open `http://127.0.0.1:5173`.

## Prototype Highlights

- Six-star holographic card reveal.
- Camera or photo-library upload with an immediate card preview.
- Demo likely-match flow that turns an uploaded photo into a rarity card.
- Physical-feeling card interactions: tilt, press, flip, showcase.
- Detailed card backs with habitat, seasonality, alternatives, privacy, and wildlife guidance.
- Rarity-based card binder.
- Friends activity built around card stacks and collection milestones.
- Privacy and wildlife-safety copy baked into the card system.
- Web Share support with a clipboard fallback.
